# Garmin Tool Kit (Garmin Connect 數據工具包)

這是一個功能強大的工具包，旨在透過 Garmin Connect Web API 自動化獲取、匯出與管理您的運動數據與訓練計畫。

## 🚀 核心功能

- **活動數據匯出 (`activity.py`)**：支援多種格式 (GPX, TCX, JSON, Original) 的批次下載，並可依日期範圍、數量進行篩選。
- **訓練計畫管理 (`workout.py`)**：提供訓練計畫的列出與上傳功能（持續開發中）。
- **靈活認證**：支援命令列參數、環境變數檔案 (`.env`) 以及互動式輸入。
- **自動化 SSO**：支援儲存會話資訊，減少重複登入次數。

## 🛠 技術棧

- **核心語言**：Python 3.10+
- **非同步處理**：AsyncIO / Garth
- **數據驗證**：Pydantic v2
- **日誌系統**：Loguru
- **進度顯示**：tqdm
- **快取雜湊**：blake3

## 📖 使用方式

### 1. 安裝依賴
請確保已安裝 Python 3.10 以上版本，並執行以下指令安裝必要套件：
```bash
pip install -r requirements.txt
```

### 2. 活動匯出工具 (`activity.py`)
用於從 Garmin Connect 批次下載您的運動記錄。

**基本範例：**
```bash
# 下載最近的 5 個活動（預設為 GPX 格式，儲存至 ./data）
python activity.py --count 5

# 從 .env 檔案讀取帳密，並下載 2023 年 8 月的所有原始檔案
python activity.py --env-file .env -sd 2023-08-01 -ed 2023-08-31 -f original
```

**CLI 參數說明：**
| 參數 | 說明 | 預設值 |
| :--- | :--- | :--- |
| `--username` | Garmin Connect 使用者名稱 (電子郵件) | 無 (提示輸入) |
| `--password` | Garmin Connect 密碼 | 無 (提示輸入) |
| `--env-file` | 指定 `.env` 檔案路徑以讀取帳密 | 無 |
| `-c`, `--count` | 下載最近活動數量，或輸入 `all` | `1` |
| `-sd`, `--start_date`| 開始日期 (格式: YYYY-MM-DD) | 無 |
| `-ed`, `--end_date` | 結束日期 (格式: YYYY-MM-DD) | 無 |
| `-f`, `--format` | 匯出格式: `gpx`, `tcx`, `original`, `json` | `gpx` |
| `-d`, `--directory` | 匯出檔案存放目錄 | `./data` |
| `-ot`, `--originaltime`| 將檔案系統時間設定為活動原始開始時間 | 關閉 |
| `--desc [LEN]` | 將活動描述附加至檔名 (可指定限制長度) | 關閉 |
| `-ss`, `--session` | 儲存/讀取會話資訊的目錄 | `.garth` |
| `-v`, `-vv` | 增加日誌詳細度 (DEBUG / TRACE) | INFO |

## 📝 檔案規範與結構

- **數據存放**：預設匯出的活動數據將存放在 `./data/` 目錄。
- **檔名格式**：`activity_YYYY-mm-dd_HH-ii-ss+時區`，確保時區資訊明確且符合生產環境標準。
- **認證快取**：SSO 資訊預設存放在 `.garth/`，有效期限內無需重新登入。
